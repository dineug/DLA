## 원시 값과 참조 값

- 원시 값 - 단순한 데이터 (undefined, null, Boolean, Number, String)
  > 변수에 저장된 실제 값을 조작: 값으로 접근한다 (Call-by-value)
- 참조 값 - 여러 값으로 구성되는 객체
  > 해당 겍체에 대한 참조를 조작: 참조로 접근한다 (Call-by-reference)

```js
const a = 24; // 원시 값
const b = []; // 참조 값
```
